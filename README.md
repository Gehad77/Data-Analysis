This repository contains a comprehensive learning roadmap for becoming a professional Data Analyst.
The roadmap is written from the perspective of a Data Analytics Manager and covers all core areas:
math & statistics foundations, Excel/Google Sheets, SQL and relational databases, Python for data analysis, 
practical statistics, data visualization with Power BI and Tableau, 
data cleaning & EDA, data warehouse & cloud fundamentals, supporting tools (Git, APIs, command line), and domain knowledge & portfolio building.

Each section includes what to learn and manager-level advice on how to study and practice using realistic datasets and business scenarios.
